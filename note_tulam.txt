git init
khởi tạo một repository Git mới trong thư mục hiện tại. Nó tạo ra một thư mục .git ẩn, nơi Git sẽ lưu trữ tất cả các thông tin về phiên bản của dự án
git add .
thêm tất cả các thay đổi trong thư mục hiện tại vào staging area. Staging area là nơi bạn chuẩn bị các thay đổi để commit
git commit -m "Your commit message"
tạo một commit mới với các thay đổi đã được thêm vào staging area. Commit là một snapshot của các thay đổi trong dự án
git push origin main
đẩy các commit từ nhánh main của repository cục bộ lên remote repository origin trên GitHub
